# IT Asset Management System
 A simple C# console application for managing IT assets within an organization.

 ## Features
 - Add assets
 - View Assets
 - Console-based menu system

 ## Technologies used
 - C#
 - .NET Console Application
 - Visual Studio code
 - Github
